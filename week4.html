What is a Browser?

The word "browser" emerged preceding the Web as a term for user interfaces that let you navigate through the page and read the text online. It lets the user look at the information they are searching for on the World Wide Web (www), this includes looking at images, web pages, and videos. A browser is used by the end-users, as it can be identified as an application program that provides interaction between the user and the internet. It uses HTTP(HyperText Transfer Protocol) which is a protocol about how information is exchanged between a client and the server, to make requests of the server throughout the internet on behalf of the user.

Web browsers have allowed us to access the websites we want to visit on the internet, this shows that the internet plays a big part in our lives. any information we want to find or access can now be found on the internet. Although Web browsers are not the same and may or may not function the same way, software designers use common features to design them, these are, a home button- which takes the user to a default homepage; a Web address bar- allowing the user to input a Web address of the website they wish to visit; back and forward buttons- which allow the user to move to the next and previous page; refresh button- to reload a Web page; stop button- to cancel what the user has searched for and end the communication with a server; tabs- allowing users to open multiple websites, and bookmarks-which allow users to select specific websites of their choice. Many browsers also have plug-ins that allow the user to add security features to their browser.

How do Browsers work?

Web browsers that we are most familiar with are Chrome, Firefox, Opera, Lynx, Safari, Edge, Flock, and Internet Explorer. The one that is used most of the time is Chrome, as it is the fastest browser. The growth in information and communication technology has made users use the internet every second, especially to promote learning and teaching.

Web browsers use the client-server model, whereby the phone browser acts as the client, being the one responsible for forwarding a request and communicating with the server, the server then sends back a response to the browser which will then construct and render the information needed by the user on the device. It is made up of integrated parts including the user interface which allows for the interaction between the browser and the user. The browser engine examines the rendering engine, and the rendering engine interprets HTML or XML documents for the requested Web page. Every Web page on the browser has its own URL( Uniform Resource Locator) which is the web address.
